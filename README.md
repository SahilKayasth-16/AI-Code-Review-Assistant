# AI Code Review Assistant

An AI-powered Code Review Assistant built using the MERN Stack.

## Features

- User Authentication
- Paste Code Snippets
- Static Code Analysis
- AI-Powered Code Review
- Review History
- Dashboard
- Secure JWT Authentication

## Tech Stack

### Frontend
- React
- Vite
- React Router
- Axios

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- ESLint
- Gemini API

## Project Structure

 - client/

## Installation
1) Frontend
 - cd client
 - npm install

2) Backend
 - cd server
 - npm install
